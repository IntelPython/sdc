# HPAT Benchmarks
Benchmarking with Airspeed Velocity.

### Install Airspeed Velocity in Anaconda Prompt:
`conda install -c conda-forge asv`

### Run benchmarking:
`cd asv_bench`<br />
`asv run`

### View results locally:
1. Run below commands:<br />
`asv publish`<br />
`asv preview`<br />
2. Open `http://127.0.0.1:8080` in a browser.
