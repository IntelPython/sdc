Pandas Window
=============
.. currentmodule:: pandas.core.window

Standard moving window functions
--------------------------------

   :ref:`Rolling.count <pandas.core.window.Rolling.count>`
       The rolling count of any non-NaN observations inside the window.
   :ref:`Rolling.sum <pandas.core.window.Rolling.sum>`
       Calculate rolling sum of given DataFrame or Series.
   :ref:`Rolling.mean <pandas.core.window.Rolling.mean>`
       Calculate the rolling mean of the values.
   :ref:`Rolling.median <pandas.core.window.Rolling.median>`
       Calculate the rolling median.
   :ref:`Rolling.var <pandas.core.window.Rolling.var>`
       Calculate unbiased rolling variance.
   :ref:`Rolling.std <pandas.core.window.Rolling.std>`
       Calculate rolling standard deviation.
   :ref:`Rolling.min <pandas.core.window.Rolling.min>`
       Calculate the rolling minimum.
   :ref:`Rolling.max <pandas.core.window.Rolling.max>`
       Calculate the rolling maximum.
   :ref:`Rolling.corr <pandas.core.window.Rolling.corr>`
       Calculate rolling correlation.
   :ref:`Rolling.cov <pandas.core.window.Rolling.cov>`
       Calculate the rolling sample covariance.
   :ref:`Rolling.skew <pandas.core.window.Rolling.skew>`
       Unbiased rolling skewness.
   :ref:`Rolling.kurt <pandas.core.window.Rolling.kurt>`
       Calculate unbiased rolling kurtosis.
   :ref:`Rolling.apply <pandas.core.window.Rolling.apply>`
       The rolling function's apply function.
   :ref:`Rolling.aggregate <pandas.core.window.Rolling.aggregate>`
       Aggregate using one or more operations over the specified axis.

        **Unsupported by Intel SDC**.
   :ref:`Rolling.quantile <pandas.core.window.Rolling.quantile>`
       Calculate the rolling quantile.
   :ref:`Window.mean <pandas.core.window.Window.mean>`
       Calculate the window mean of the values.

        **Unsupported by Intel SDC**.
   :ref:`Window.sum <pandas.core.window.Window.sum>`
       Calculate window sum of given DataFrame or Series.

        **Unsupported by Intel SDC**.
